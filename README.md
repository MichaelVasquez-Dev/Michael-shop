# Michael API

1. Clonar proyecto

2. ejecutar el comando ```npm install ``` o ```yarn install```

3. Clonar el archivo ```.env.temp``` y renombrarlo a ```.env```

4. Cambiar las variables de entorno

5. Levantar la base de datos
```
docker-compose up -d
```

6. Levantar modo desarrollo ``` yarn start:dev ```

7. Ejecutar Seed
```
http://localhost:3000/api/seed
```
